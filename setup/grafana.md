# **Setup Grafana**

## **Download Grafana for Window:**

`https://grafana.com/grafana/download`

### **Start grafana**

`brew services start grafana`

### **Stop grafana**

`brew services stop grafana`

## **Connection MySQL DB:**

```
Visit: http://localhost:3000
```
